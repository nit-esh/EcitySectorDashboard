| Context | Opening Used |
|---|---|

## 3. Sign-offs

| Context | Sign-off Used |
|---|---|
| Isha / spiritual context | "Pranam" |
| Quick replies | None (message ends directly) |
| Warm personal note | "Thank you" |

Key pattern: Never "Regards", "Best", "Warm regards", or "Cheers."
